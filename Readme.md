# Quizbot
Simple quizbot!

## Requirements
Python 3.6+, PostgreSQL, Redis

## Installation
Install python libraries
```shell
$ pip3 install -r requirements.txt
```

Copy `.env.dist` file
```shell
$ cp .env.dist .env
```
Write the Environment variables to the `.env` file
```shell
$ nano .env
```

Run app
```shell
$ python3 app.py
```
